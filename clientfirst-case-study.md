# Client First Investments

### Website Optimization & Lead Generation Case Study

---

## Project Summary

**Client:** Client First Investments
**Industry:** Property Investment
**Project Type:** Website Optimization & Lead Generation Setup
**Focus:** Messaging clarity, conversion improvements, and paid traffic readiness
**Tools Used:** Wix, Google Ads, GitHub

---

## Project Overview

This project focused on improving the clarity, structure, and conversion potential of the Client First Investments website while preparing the business for scalable lead generation.

The objective was to simplify messaging, improve user experience, strengthen calls-to-action, and create a clear pathway from website visit to consultation booking.

---

## Key Improvements

### 🔹 Website Optimization

• Improved headline messaging
• Clarified value proposition
• Simplified service explanations
• Added a structured process section
• Improved testimonial positioning
• Strengthened consultation call-to-action

---

### 🔹 Service Page Improvements

The following pages were optimized to improve readability and conversion:

• Tax Minimisation
• Debt Reduction
• Wealth Creation
• Coaching & Mentoring

Each page was restructured with:

• Clearer headlines
• Improved readability
• Benefit-driven messaging
• Stronger calls-to-action

---

### 🔹 Landing Page Development

A dedicated landing page was created to support lead generation campaigns.

The page was structured to guide visitors from awareness to consultation booking using:

• Clear messaging
• Strong value proposition
• Step-by-step process
• Social proof
• Focused call-to-action

👉 [View Landing Page Project](09-landing-page-project/landing-page-final.md)

---

### 🔹 Google Ads Setup

A Google Ads account was created and configured to support paid lead generation.

Setup work included:

• Account creation and billing setup
• Google tag installation on Wix
• Conversion tracking configuration
• Thank-you page setup for tracking
• Form redirect implementation

---

### 🔹 Conversion Tracking Setup

A structured tracking system was implemented to accurately measure consultation form submissions.

Conversion flow:

1. Visitor lands on the consultation landing page
2. Visitor submits the form
3. Visitor is redirected to a thank-you page
4. The thank-you page visit is recorded as a conversion

This created a reliable and measurable funnel for campaign performance.

---

### 🔹 Campaign Creation

A Google Search campaign was created to promote the Free Property Investment Consultation.

Campaign details:

• Campaign type: Search
• Objective: Leads
• Daily budget: A$20
• Location targeting: Australia
• Language: English
• Bidding strategy: Maximize conversions
• Keyword targeting focused on property investment consultation

The campaign is currently in the learning phase while performance data is being gathered.
